# E-Commerce Data Querying & SQL Insights (Week 3)

This repository contains the SQL scripts, dataset, and documentation for the Week 3 analytics project. The objective was to write structured SQL queries to filter, group, aggregate, and extract insights from the transaction database.

## Repository Contents

The following files are included in this project:

| File Name | Format | Description |
| :--- | :---: | :--- |
| `queries.sql` | SQL Script | A fully-commented SQL file containing schema definitions and analytical queries. |
| `Dataset for Data Analytics p3.xlsx` | Excel Workbook | The primary transaction dataset representing the database table contents. |
| `Data Analytics Project 3.pdf` | PDF Document | The original project assignment sheet. |

## SQL Script Overview

The `queries.sql` script is divided into sections covering:

1. **Table Schema Creation**:
   * Setup code to build the `transactions` table in a standard SQL database (such as SQLite, MySQL, or Postgres).

2. **Core Statistics**:
   * Uses `COUNT`, `SUM`, `AVG`, `MIN`, and `MAX` to compute gross revenue, average order value, average quantity, and cart sizes.

3. **Product performance**:
   * Groups records by `Product` and ranks them by total revenue to find the best-selling inventory lines.

4. **Fulfillment funnel analysis**:
   * Groups by `OrderStatus` to calculate the volume and percentage of cancelled, returned, pending, and shipped orders.

5. **Sales Trends & Seasonality**:
   * Groups by Year and Month using date functions (`STRFTIME`) to show monthly seasonality and H1 year-over-year revenue comparison.

6. **Marketing & Referral Attribution**:
   * Identifies total orders and revenue generated by referral sources (Instagram, Email, Google, Facebook, Referral).

7. **Coupon Code & Returns Correlation**:
   * Uses conditional aggregation (`CASE WHEN`) to evaluate how different coupon codes affect customer return rates.

8. **Outlier Detection**:
   * Extracts high-value transactions that lie beyond the statistical IQR upper bound ($3,330.41).
